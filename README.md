# SMASH ARENA

Canvas で動くブラウザ対戦アクションゲームです。キャラクター選択、1P vs CPU、2P 対戦、ストック制、ダメージ%によるノックバック、ガード、複数攻撃、BGM/SE を備えています。

## 起動方法

このリポジトリを取得したあと、ローカルサーバーで配信してください。

```powershell
python -m http.server 8127 --bind 127.0.0.1
```

ブラウザで開きます。

```text
http://127.0.0.1:8127/index.html
```

画像アセットを読み込むため、`index.html` を直接ダブルクリックするより HTTP サーバー経由の起動を推奨します。

## 操作

1P:

- 移動: `A` / `D`
- ジャンプ: `W` / `Space`
- しゃがみ / 降り: `S`
- 弱攻撃: `J`
- 強攻撃 / スマッシュ: `K`
- 必殺 / 飛び道具: `L`
- ガード: `H` / `Shift`

2P:

- 移動: `←` / `→`
- ジャンプ: `↑`
- しゃがみ / 降り: `↓`
- 弱攻撃: `.`
- 強攻撃 / スマッシュ: `/`
- 必殺 / 飛び道具: `,`
- ガード: `Right Shift` / `Right Ctrl`

共通:

- ミュート: `M`
- 一時停止: `P`
- 一時停止中にキャラ選択へ戻る: `R`

## 現在の内容

- 8体のオリジナルファイター
- 1P vs CPU / 2P 対戦
- CPU 難易度切り替え
- ストック数切り替え
- リアル寄りの山岳背景
- 浮遊遺跡風ステージ
- アニメ調キャラクター描画
- WebAudio ベースの高品質サウンドミキサー
- リバーブ、コンプレッサー、低音レイヤーつき SE
- サンプル音源SE（Kenney CC0）+ Sunoバトル BGM（未ロード時は合成音へ自動フォールバック）
- ヒットフラッシュ / 衝撃リング / 足場発光 / 重量級着地エフェクト
